HTML Lesson Notes HTML - Hyper Text Markup Language HTML is the standardized system for formatting the text documents that make up web pages. Most of HTML is regular text, but then there are tags and elements. Tags tell the browser how to display the elements of the web page. From formatting the text, like bold or italic, to adding images, videos and hyper links.
Tags
Tags are instructions to the browser on how to display the data in the html document. There are many tags, tags to bold text, italicize, center, line break.

Inline vs Block Elements
HTML elements are either inline, just text entered like in a notepad or word doc, or Block, which puts the element into a small invisible box.
